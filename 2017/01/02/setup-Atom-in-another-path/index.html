解决Atom安装不提示路径问题

发表于 2017-01-02 | 分类于 教程

记录Atom安装不提示路径直接安装在C盘问题的解决

最近看一个大牛在博客评论Atom,甚是有好感,遂装了一个,结果安装的时候出现问题了:

从官网上下载的Atom,安装时双击之后发现并没有任何提示选项,过了一段时间之后直接打开了….打开了…双击之后直接默认安装了,不用说,肯定是安装在C盘了,不能忍!!!

在网上扒了一段时间发现可以这样解决:

在官网下载Atom的时候不要选择Download Windows Installer,而是选择下面的Other platforms,然后在打开页中选择对应的zip包,下载之后解压使用,这样就可以随意控制路径了.
但是,这样做安装包之间是否有区别还有待考证.
